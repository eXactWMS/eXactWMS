O que é o eXactWMS?

Um software para Gerenciamento de Armazéns (Centro de Distribuição) que oferece UI e UX moderna, rastreabilidade, eficiência para Supply chain.

Funciona somente no Windows?

Os client (terminal) são desktop, porém temos versão Mobile para SO Android, e a parte de retarguarda (API´s e Banco de Dados) podem ser instalados em SO Windows, Linux e com Docker.

Quais as principais funcionalidades do eXactWMS?

Planejamento, Agendamento(Em breve), Recebimento, Armazenamento, Separação, CheckOut, Expedição, Transporte. Todas as funcionalidades podem serem executadas no coletor de dados(RF) Android.
